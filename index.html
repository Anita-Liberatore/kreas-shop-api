Business rule

Discount logic

10%
Cart discount

When the customer adds more than 3 products to the cart, a 10% discount is automatically applied to the total.

const total = cart.reduce(
  (sum, item) => sum + item.price, 0
)

const discount = cart.length > 3
  ? total * 0.10
  : 0

const finalTotal = total - discount
// computed properties
const total = computed(() =>
  cart.value.reduce((s, i) => s + i.price, 0)
)

const discount = computed(() =>
  cart.value.length > 3 ? total.value * 0.10 : 0
)

const finalTotal = computed(() =>
  total.value - discount.value
)
Integration

Fetch the data

Drop this into your Vue.js component and start building.

const API = 'https://raw.githubusercontent.com/
  Anita-Liberatore/kreas-shop-api/master/products.json'

const getProducts = async () => {
  const res = await fetch(API)
  return res.json() // Array of 24 products
}
// Vue 3 — Composition API
import { ref, onMounted } from 'vue'

const products = ref([])

onMounted(async () => {
  const res = await fetch('https://raw.githubusercontent.com/
    Anita-Liberatore/kreas-shop-api/master/products.json')
  products.value = await res.json()
})
// Axios
import axios from 'axios'

const { data: products } = await axios.get(
  'https://raw.githubusercontent.com/
  Anita-Liberatore/kreas-shop-api/master/products.json'
)
// products → Array of 24 items
